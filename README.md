# MyReads Project

A book shelf react application for beginner.

How to install: Git clone this project, and go into project-myreads-udacitty folder, type:


* install all project dependencies with 

```js
npm install
```

* start the development server with

```js
npm start
``` 

* start in browser

```js
localhost:3000
```
